# ITAM Borrow System Redesign

ระบบ ITAM Borrow ได้รับการออกแบบส่วนติดต่อผู้ใช้งาน (UI) ใหม่ทั้งหมดให้มีความหรูหรา ทันสมัย และเป็นไปตามมาตรฐานการออกแบบเว็บแอปพลิเคชันยุคปัจจุบัน โดยอ้างอิงและปรับปรุงจากต้นแบบของระบบ ITSM

## รายละเอียดการปรับปรุงดีไซน์ (Full System Redesign)

1. **ระบบโทนสีและการจัดวาง (Layout & Color System)**
   - เปลี่ยนแปลงสีพื้นหลังหลักของหน้าเว็บทั้งหมดเป็นสีโทนสว่างสบายตา `bg-slate-50`
   - กำหนดรูปแบบการ์ดแสดงผลทั้งหมดเป็นสีขาวมนโค้ง `bg-white rounded-2xl border-slate-200`
   - เลือกใช้สีฟ้า `sky-500` (Sky Blue) สำหรับปุ่มหลัก, ลิงก์ที่ต้องการเน้น, และไอคอนไฮไลท์ เพื่อส่งมอบความพรีเมียม
   - เลิกใช้งานป้ายสัญลักษณ์ Material Design 3 (M3 Tokens) ตัวเดิมที่ซับซ้อนใน `globals.css` โดยหันมาใช้ระบบสีมาตรฐานของ Tailwind CSS

2. **หน้าล็อกอินระบบ (Login Page - `/login`)**
   - ออกแบบใหม่ทั้งหมดด้วยเลย์เอาต์ **Split Screen (แบ่งครึ่งจอ)** ในหน้าจอเดสก์ท็อป
     - **ฝั่งซ้าย:** พื้นหลังไล่ระดับเฉดสี (Gradient) สวยงาม จาก `sky-500` ไปยัง `indigo-700` พร้อมข้อความแนะนำระบบ แถบตกแต่ง และข้อมูลไอคอนสรุปจุดเด่นของระบบ
     - **ฝั่งขวา:** ฟอร์มล็อกอินสไตล์คลีนการ์ดสีขาวแบบมน พร้อมปุ่ม Microsoft Single Sign-On (SSO) และปุ่มทดสอบสิทธิ์ผู้ดูแลระบบแยกสัดส่วนชัดเจน
   - หน้าจอสมาร์ทโฟนจะแสดงฟอร์มแบบกึ่งกลางที่เน้นการใช้งานง่าย

3. **หน้ารายการอุปกรณ์สำหรับพนักงาน (Assets Catalog - `/assets`)**
   - เปลี่ยนรูปแบบรายการยืมอุปกรณ์ให้มีอารมณ์คล้าย **Catalog/Marketplace**
   - การ์ดอุปกรณ์มีลูกเล่นเอฟเฟกต์สีขอบและเงาเมื่อชี้เมาส์ (Hover Effect)
   - แถบค้นหาอุปกรณ์ขนาดใหญ่โดดเด่นสะดุดตา
   - ปุ่มเลือกคัดกรองตามหมวดหมู่ถูกเปลี่ยนเป็นแบบกล่องมนเม็ดยา (Pills) ที่สามารถเลื่อนซ้าย-ขวาได้แบบลื่นไหล
   - ไอคอนถูกเปลี่ยนเป็น `lucide-react` เพื่อให้สอดรับกับแต่ละหมวดหมู่ (เช่น โน้ตบุ๊กเป็น `Laptop`, เมาส์เป็น `Mouse`, เครื่องพิมพ์เป็น `Printer`)
   - ปุ่มสถานะ (Badge) ออกแบบใหม่ให้มีเส้นกรอบบางและสีสถานะโปร่งแสง (เช่น ว่าง = เขียว, ยืมอยู่ = น้ำเงิน)

4. **หน้าข้อมูลส่วนตัว (Profile Page - `/profile`)**
   - ดีไซน์ใหม่ให้มีกล่องหัวข้อโปรไฟล์ขนาดใหญ่พร้อมรูปวงกลมมนขอบสีฟ้า
   - เพิ่มบล็อกสรุปสถิติ (Stats Grid) การใช้งาน เช่น "รายการที่กำลังยืมอยู่" และ "จำนวนการยืมทั้งหมด"
   - รายการการตั้งค่าอื่นๆ ถูกแยกย่อยออกเป็นปุ่มกดที่มีเอฟเฟกต์ตอบรับแบบเคลื่อนไหว และปุ่มออกจากระบบสีแดงเด่นชัด

5. **เมนูนำทางด้านล่างสำหรับมือถือ (Bottom Navigation - `bottom-nav.tsx`)**
   - ปรับใช้ดีไซน์กล่องเมนูกึ่งโปร่งแสง (Glassmorphism) โค้งมนพร้อมเงาฟุ้งสะท้อน
   - เปลี่ยนไอคอนมาเป็นรูปแบบไอคอนดีไซน์เรียบหรูของ `lucide-react`
   - เพิ่มสัญลักษณ์จุดกลมขีดสีฟ้าด้านใต้ไอคอนสำหรับสถานะหน้าเพจที่กำลังเปิดใช้งาน (Active Indicator Dot) พร้อมแอนิเมชันตอนสลับหน้า

6. **เมนูนำทางด้านข้างของเดสก์ท็อป (Sidebar Navigation - `side-nav.tsx`)**
   - ออกแบบการจัดวางเมนูด้านซ้ายให้เหมาะสมกับจอคอมพิวเตอร์เดสก์ท็อป
   - เมนูที่กดใช้งานจะมีแถบขีดสีฟ้าที่ขอบซ้าย (Active Left Border) พร้อมพื้นหลังสีฟ้าบางเพื่อความง่ายในการแยกแยะ
   - ส่วนท้ายของเมนูด้านซ้ายแสดงรายละเอียดข้อมูลส่วนตัวและรูปโปรไฟล์ พร้อมปุ่มออกจากระบบที่เรียกใช้งานได้ทันที

7. **ส่วนประกอบกลางอื่นๆ (Global Components Redesign)**
   - **ConfirmModal:** ปรับแต่งการตอบรับในรูปแบบแอนิเมชันสไลด์ขึ้นบนมือถือ และเฟดอินบนเดสก์ท็อป ใช้ปุ่มดีไซน์ใหม่ทดแทนแบบเดิม
   - **StatusBadge & SearchInput & FilterPills & Header:** แปลงรหัสสีไอคอนและรูปแบบเดิมให้เป็นตามระบบ Slate CSS ใหม่ทั้งหมด
   - **AssetForm:** หน้าจอฟอร์มเพิ่ม-แก้ไขอุปกรณ์ไอทีได้รับการจัดวางกลุ่มข้อมูลออกเป็นกล่องส่วนงานชัดเจน (ข้อมูลระบุตัวตน, สถานะการใช้งาน, ข้อมูลการจัดซื้อ, ตำแหน่งที่ตั้ง)
